AR Glasses System is a smart wearable prototype that combines ESP32, Bluetooth Low Energy (BLE), and an Android application to provide real-time information directly on a near-eye display. The project demonstrates how IoT and mobile technologies can work together to create a lightweight augmented reality experience.

The Android application communicates with the ESP32 over BLE, enabling seamless data exchange between the smartphone and the wearable device. The system is designed to display useful information without requiring users to constantly check their phones.

Features
📱 Bluetooth Low Energy (BLE) communication between ESP32 and Android
🔔 Real-time notification mirroring
🧭 Navigation assistance with turn-by-turn directions
💡 Adjustable display brightness
🔋 Battery level monitoring
📲 Android application for device control
⚡ Low-power wearable communication
🖥️ OLED display integration
